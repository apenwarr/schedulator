MobWrite Libraries

These files are shared by different flavours of MobWrite.

diff_match_patch.*
The Diff Match Patch Library that forms the heart of MobWrite's differential
synchronization.  http://code.google.com/p/google-diff-match-patch/

default_editor.html
Page to load when loading q.py with an in-line id.

mobwrite_config.txt
Settings to control a server's behaviour.

mobwrite_core.py
Common functions shared between the Daemon and App Engine servers.

mobwrite_core_test.py
Unit tests for mobwrite_core.py.
